# Personal Portfolio Website

🚀 A responsive personal portfolio website to showcase my skills, projects, certifications, achievements, and resume.

## 📖 Overview

This portfolio website serves as a digital resume and professional profile. It highlights my technical skills, academic projects, certifications, and contact information.

- About Me Section
- Skills Showcase
- Projects Gallery
- Certifications Section
- Resume Download
- Contact Form

- JavaScript
